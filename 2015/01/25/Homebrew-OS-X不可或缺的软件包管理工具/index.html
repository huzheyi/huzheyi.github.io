Homebrew - OS X不可或缺的软件包管理工具

By huzheyi

Jan 25 2015 Updated:Jan 26 2015

Contents

  2. 2. Homebrew的使用
  3. 3. Homebrew的更新

时下，Linux类操作系统都有软件包管理工具，例如：RHEL/Centos的yum，Debian/Ubuntu的apt，Arch的pacman等，用来安装、卸载、更新软件包十分方便。如今，Mac OS X下除了官方的App Store，也拥有了一个强大的软件包管理工具——Homebrew(Github)。

Homebrew的安装

Homebrew的安装十分方便，仅需在终端窗口中输入一行命令：

ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"

Homebrew的使用

Homebrew的原则是“No Sudo”，即再也不需要Sudo来安装删除软件包了。下面是Homebrew的用法，十分简单(以wget为例)：

  • 查找软件包

    brew search wget

  • 安装软件包

    brew install wget

  • 列出已安装的软件包

    brew list

  • 删除软件包

    brew remove wget

  • 查看软件包信息

    brew info wget

  • 列出软件包的依赖关系

    brew deps wget

  • 列出过时的软件包

    brew outdated

  • 更新过时的指定软件包

    brew upgrade wget

  • 更新过时的全部软件包

    brew upgrade

Homebrew的更新

Homebrew的更新，同样十分方便：

brew update

自定义软件包管理

如果需要的软件包不能再Homebrew找到，同时又想通过Homebrew来方便的管理，那么久可以使用Homebrew来自定义软件包。

  1. 找到软件源码地址

    http://xxx.com/source-1.0.tgz

  2. 建立自己的formula

    brew create http://xxx.com/source-1.0.tgz
    Homebrew自动建立的formula已包含configure和make install，一般不需要进行修改，退出编辑安装即可。

  3. 如果有需要编辑formula

    brew edit source

  4. 安装自定义软件包

    brew install souce
